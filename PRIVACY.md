# Privacy Policy — Webflow Class Guard

**Last updated: March 2026**

Webflow Class Guard is developed by [Paper Tiger](https://papertiger.com).

---

## What data we collect

None. Webflow Class Guard does not collect, transmit, or share any personal data or usage information.

---

## What the extension stores locally

Webflow Class Guard stores the following in Chrome's built-in `chrome.storage.sync`:

- The list of CSS class names you have chosen to protect, scoped per Webflow project
- Your selected framework preset (MAST or Lumos), scoped per Webflow project
- Your global on/off preference

This data exists solely on your device (and synced across your own Chrome profile if you are signed into Chrome). It is never sent to any server, and Paper Tiger has no access to it.

---

## What websites the extension accesses

Webflow Class Guard only activates on the following URLs:

- `https://*.design.webflow.com/*`
- `https://webflow.com/design/*`

It does not run on any other website. It does not read page content, form data, cookies, or any information from the Webflow projects you work on. It only reads the CSS class names visible in the Designer's selector bar in order to apply protection.

---

## Third-party services

None. The extension makes no network requests and does not load any third-party scripts or resources.
